## Focus areas

- **Knowledge Graphs** — connected representations of software systems: services, APIs, databases, teams, decisions. Capture both structure and intent.
- **GraphRAG** — retrieval that combines semantic similarity with structural traversal. The layer where most agent systems either earn their answers or fail silently.
- **Agent Systems** — coordinated agents for engineering workflows (code navigation, architecture reasoning, documentation discovery, PR review) that share retrieval but specialize in reasoning.
- **Developer Tooling** — surfaces, interfaces, and APIs that put architectural context in front of engineers at the moment they need it.
- **Intelligent Architectures** — end-to-end pipelines from raw data (source code, documentation, telemetry) to grounded, inspectable answers.

---

## Engineering principles

- **Architecture over features.** The shape of the system determines what it can do.
- **Retrieval is the substrate.** Reasoning is the layer above. Get retrieval wrong and reasoning becomes theater.
- **Tradeoffs over abstractions.** Every abstraction is a bet about the future; the winning bet is the one that admits what it doesn't know.
- **Build for the question no existing tool can answer.** The simple cases will follow.

## Currently exploring

- End-to-end GraphRAG pipelines where entity extraction, relationship inference, and traversal are evaluated against ground-truth architecture questions rather than vector-similarity benchmarks.
- Agent memory patterns: episodic vs. semantic memory, the boundary between retrieval and reasoning, and when each is the right primitive.
- Context engineering over prompt engineering. The shape of the context window matters more than clever prompting; the leverage is in what the agent sees, not what it is told.
- Knowledge-graph evaluation methodologies that hold up outside toy corpora — particularly for code, where the graph is dense and ground truth is partial.

Notes and working drafts at [sunilskcj.in/research](https://sunilskcj.in/research) when they are ready to share publicly.
